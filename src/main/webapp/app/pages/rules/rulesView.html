DevWars Official Rules

DevWars is still relatively new. As the game progresses, our rules will mature and become a lot more solid than they are now. Notice any inconsistencies? Let us know!

General Rules
  • You are only allowed to code in the language you are assigned to.
  • The DevWars Twitch channel must not be open whilst competing.
  • No copying and pasting code, however, copying code from teammates is acceptable.
  • Copying and pasting URLs is permitted.
  • Use of the console in C9 is prohibbited.
  • After the 60 minute timer has run out, your code can no longer be modified.
  • SASS and SCSS are allowed for CSS players.
Classic Rules
  • HTML, CSS, JavaScript are allowed.
  • Only Whitelisted Libraries & Frameworks (see below) are allowed, but copying snippets from them is not.
  • Inline styling is not allowed. All styling must be performed using CSS only. Javascript may use CSS only for dynamic purposes.
Scoring [Max Points = 10]
  • Objectives
    There are 4 objectives and 1 bonus objective. Each objective is worth [1] Point (including the bonus objective). The only way to earn the point from the bonus objective is to complete all 4 objectives.
  • Design Vote (UI)
    Viewers can vote for which website they think has the best design. You can earn up to [2] points if you earn more than 66% of votes.
  • Functionality Vote (UX)
    Viewers can vote for which website they think has the best functionality. You can earn up to [2] points if you earn more than 66% of votes.
Zen Garden Rules
  • Only CSS is allowed.
  • You are not allowed to modify the HTML template that is given to you.
Scoring [Max Points = 4]
  • Design Vote (UI)
    Viewers can vote for which website they think has the best design. You can earn up to [2] points if you earn more than 66% of votes.
  • Functionality Vote (UX)
    Viewers can vote for which website they think has the best functionality. You can earn up to [2] points if you earn more than 66% of votes.
